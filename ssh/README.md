# SSH Configuration Project

This project automates SSH setup and configuration tasks.

## Files

- **0-use_a_private_key**: Connects to a remote server using the SSH private key.
- **1-create_ssh_key_pair**: Generates a new RSA key pair with a passphrase.
- **2-ssh_config**: SSH client configuration file disabling password authentication.
- **100-puppet_ssh_config.pp**: Puppet manifest to configure SSH client options.

All files are executable and follow ALU guidelines.
